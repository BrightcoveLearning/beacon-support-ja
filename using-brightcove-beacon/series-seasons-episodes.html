概要

シリーズは、通常、共通のテーマを共有する特定の順序で表示される関連ビデオのコレクションです。シリーズを作成するときは、シリーズ、その季節、および関連するエピソード間の関係を定義する必要があります。

シーズンとは、設定された期間をカバーする動画のグループです。各シリーズには、1 つ以上の季節を指定できます。

エピソードは、個々のビデオです。これは特定の映画と考えることができます。

シリーズと季節を定義し、エピソードを摂取すると、それらを Brightcove Beacon アプリケーションで使用できます。これを行うには、次の手順を実行します。

  1. シリーズ（Brightcove Beacon ）

    Brightcove Beacon で、シリーズを作成します。ここでは、シリーズの名前を指定し、在庫状況や画像などの情報を提供します。

  2. シーズンズ (Brightcove Beacon )

    Brightcove Beacon で、シリーズに関連する季節を作成します。シーズンが属するシリーズを選択することに加えて、シリーズに類似した情報を提供します。

  3. エピソード (ビデオクラウド)

    Brightcove Beacon のエピソードコンテンツの真実は、Video Cloud にあります。

    Video Cloud Studio で、カスタムフィールドに指定された値を使用して、ビデオエピソードを対応するシーズンに割り当てます。これらのカスタムフィールドは次のとおりです。

    • beacon.episode.number
    • beacon.episode.seasonNumber
    • beacon.episode.serieName

  4. Brightcove Beacon に戻り、動画エピソードを取り込みましょう。

シリーズの作成

シリーズを作成するには、次の手順に従います。

  1. Brightcove Beacon にログインし、[ プログラム ] を選択します。
  2. 左側のナビゲーションで、[ シリーズ ] を選択します。
  3. [ 新しいシリーズを追加 ] ボタンをクリックします。

  4. [ 基本データ ] タブで、次のように入力します。

    フォームフィールド名 説明/要件
    名前 シリーズの識別子、顧客向けの名前は、後のテキストデータセクションに記載されています
    製作年 シリーズが制作された年。この情報は [ 詳細情報 ] 画面に表示されます。
    製作国 シリーズが制作されている国。この情報は [ 詳細情報 ] 画面に表示されます
    ジャンル シリーズのジャンル。必要に応じてジャンルを作成できることに注意してください。
    サブジャンル 完了しない、このデータは使用されません
    ジャンル言語プレビュー プレビューの言語
    エピソードのソートタイプ アプリのユーザーインターフェイスでエピソードを並べ替える方法
    キャスト CAST の下のメタデータが [ 詳細情報 ] 画面に表示されます。
  5. [ テキストデータ ] タブをクリックします。
  6. 記入してください タイトル （これはシリーズの視聴者向けの名前です）、 説明 かつ あらすじ。このデータは Brightcove Beacon アプリに表示されます。

  8. 次のイメージを指定します (以下の要件に一致します)。

  12. プロバイダを選択します。ほとんどの場合、Brightcove Video Cloud アカウントを参照するプロバイダーは 1 つだけです。

エピソードの取り込み

シリーズの作成と関連シーズンは Brightcove Beacon で実行されますが、シリーズのエピソードの作成は Video Cloud Studio で行われます。各エピソード動画のカスタムフィールドでシリーズとシーズンに関する値を割り当てて、それらの動画を Brightcove Beacon に取り込みます。

BrightcoveBeaconエピソードコンテンツの真実はVideoCloudにあることを忘れないでください。Brightcove Beaconでエピソードに加えた変更は、VideoCloudからコンテンツを取り込むときに上書きされます。

前提条件

次の手順に進む前に、次のことを完了していることを確認してください。

  1. ビデオクラウドスタジオにログインします。

  2. ページ上部にあるドロップダウンメニューから、[ アップロード ] を選択します。

  3. エピソードの動画をアップロードします。

  4. ページの上部にあるドロップダウンメニューから、[ MEDIA ] を選択します。

  5. 画像とその仕様書に記載されているように、エピソードのビデオ画像を更新します。

  6. ナビゲーションヘッダーで、[ 管理 ]、[ ビデオフィールド ] の順にクリックします。

    アクセスするには、アカウント管理者である必要があります管理者メニュー。
  7. 動画を Brightcove Beacon に取り込むのに必要なカスタム項目を作成します。詳細については、「 Video Cloud カスタム項目の作成」ドキュメントを参照してください。

ステップ

エピソード動画を Brightcove Beacon に取り込むには、次の手順に従います。

  1. ビデオクラウドスタジオに戻ります。
  2. ページの上部にあるドロップダウンメニューから、[ MEDIA ] を選択します。

  3. エピソードの動画のタイトルをクリックして、動画のプロパティページに入ります。

  4. 右側に [ カスタムフィールド] セクションが表示されるまで下にスクロールします。
  5. [編集]ボタンをクリックします。
  6. エピソードに関連付けられている 3 つのカスタムフィールドを見つけます。
  7. エピソードビデオの場合は、対応するカスタムフィールドにエピソード番号、シーズン番号、シリーズ名を入力します。
  8. ビデオに適切な値を入力したら、必ず [ 保存 ] ボタンをクリックします。
  9. すべてのエピソード動画について、カスタムフィールドに値を繰り返します。
  10. Brightcove Beacon にログインします。
  11. トップメニューの [ ツール ] オプション ( レンチアイコン ) をクリックします。

  12. 左側のメニューで [ 取り込み ] オプションが選択されていることを確認します。

  13. [ Brightcoveの動画を更新 ] ボタンをクリックして、新しく設定されたエピソード動画を取り込むことができます。

  14. アプリをすぐに利用できるようにするには、左側のナビゲーションの [キャッシュ] オプションをクリックして Brightcove Beacon のキャッシュをクリアし、[ キャッシュを削除 ] ボタンをクリックします。

    Brightcove Beacon ツールオプションの使用の詳細については、を参照してください。ツールの使用資料。

これで、シリーズの作成と、それに関連するシーズンとエピソードの情報は終了です。
