ツールの使用

このトピックでは、Brightcove Beacon の [ ツール] セクションにあるオプションの機能を学習します。

ツールのご紹介

Brightcove Beacon では、レンチアイコン (1) はツールのセットを表します。ツールのオプションは、取り込みとキャッシュです（2）。このドキュメントでは、2 つのオプションについて説明します。

取り込み

取り込みグループには、次の 3 つのツールがあります。

  • Brightcoveの動画を更新する :Video Cloud カタログに動画を追加した場合、または既存の動画を変更した場合、このボタンをクリックすると、新しく追加または更新された動画が Brightcove Beacon に同期されます。
  • Brightcoveのプレイリストの更新 :Video Cloud カタログにプレイリストを追加した場合、または既存のプレイリストを変更した場合、このボタンにより、新しく追加または更新されたプレイリストが Brightcove Beacon に同期されます。
  • XML EPG-フォルダから XML をフェッチする :EPG (電子プログラムガイド) を更新する必要がある場合、このボタンをクリックすると、更新された XML ファイルが取得されます。ファイルをドロップする S3 アカウントが提供されます。
また、Brightcoveのサポート担当者と協力して、取り込みを自動的に実行するスケジュールを作成することもできます。

摂取に関する注意事項

Studio の特定の動画について入力されたすべてのデータが、自動的にBrightcove Beacon に取り込まれるわけではありません。たとえば、次のものは Video Cloud から Brightcove Beacon に取り込まれません。

  • 利用可能時間 :で設定されたスケジュール時間 可用性 ビデオのプロパティのセクションは、Brightcove Beacon に取り込まれません。日付はになりますが、時刻ではありません。
  • 地理的フィルタリング :で設定された地理フィルタリング情報 可用性 ビデオのプロパティのセクションは、Brightcove Beacon に取り込まれません。
上記の問題は、を作成することで解決できます。パッケージ Brightcove Beaconでは、カスタムフィールドを介して取り込み時にそのパッケージにビデオを割り当てます。を参照してくださいサブスクリプションパッケージの作成とサブスクリプションパッケージの使用詳細については、ドキュメントを参照してください。

また、Brightcove Beacon に記載されているすべての項目にカスタム項目が関連付けられているわけではありません。例えば 製作国 かつ サブジャンル 対応するカスタムフィールドが関連付けられていません。

[キャッシュ]

[キャッシュ] (Cache) グループには 2 つのツールがあります。

  • キャッシュパージ :Brightcove Beacon で変更を行った場合、このボタンをクリックして、変更を現在の設定にロールします。このオプションは、イメージ以外のすべての変更を現在の設定に引き出します。
  • イメージキャッシュ :Brightcove Beacon の画像を更新する場合は、このボタンをクリックして、イメージの変更を現在の設定にロールします。

このプロセスは、本番環境で頻繁に行うことがスケーラビリティとパフォーマンスに悪影響を及ぼすため、無差別に実行すべきではありません。
