このライブ広告の構成プロセスは、ライブチャネルがSSAIで有効になっている場合にのみ必要です。そうでない場合、これらの手順は必要ありません。

前提条件

  • VideoCloudでSSAIを有効にしたライブチャンネルを作成します。
  • VideoCloudで広告タグを作成して設定します。
  • Brightcove Beacon でチャネルを作成します。
  • Brightcove Beacon でアドバタイズメント構成を作成します。
  • チャネルで権利とスケジュールを設定します。

Brightcove Beacon を使用したビデオクラウド広告の構成

  1. Video Cloud Studioで、住むモジュール。
  2. BrightcoveBeaconで使用するライブSSAIチャネルを作成または開きます。
  3. の中に制御室、 クリック公開して埋め込む左パネルから。
  4. の中にライブプレーヤーオプションセクションで、ドロップダウンからを選択します広告構成設定したい。
  5. 新しいものを選択したら標準の埋め込みコード現れる。
  6. を抽出しますadConfigId画像に示すように、adConfigIdに割り当てられているURLの一部をコピーします。

Brightcove Beacon で広告を作成する

  1. BrightcoveBeaconプラットフォームに戻ります。
  2. クリック広告。
  3. 設定する広告構成を作成または選択します。
  4. 新しい広告の場合は、名前。
  5. 必ずプロバイダーに設定されていますブライトコーブSSAI。
  6. の中にライブ構成IDフィールドに貼り付けますadConfigIdこのドキュメントの前のセクションでコピーしたURL。
  7. クリック新しい広告を作成する。

ブライトコーブ Beacon での権利の設定とスケジューリング

  1. 上部のナビゲーションで、をクリックしますチャンネル。
  2. SSAI構成を設定するチャネルを選択します。
  3. [ 権限とスケジュール ] タブをクリックします。
  4. の権利の種類、 選択する広告-AVOD。
  5. の広告の構成、ドロップダウンから、作成したばかりの広告構成を選択します。
  6. クリック新しいチャネルを作成するあなたの仕事を保存します。
