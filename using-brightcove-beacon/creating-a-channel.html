説明

チャンネルは、通常のTVチャンネルと同じように、24時間年中無休のライブイベントをストリーミングするために作成されます。チャンネルのコンテンツは、チャンネルが利用可能である限りブロードキャストされる一連のVODを介して、またはライブイベントを直接送信することによって生成できます。ライブイベントの作成方法の詳細については、次のWebサイトをご覧ください。 Brightcove Beacon Live Documentation。

従来の Beacon チャネルに加えて、クラウドプレイアウトチャネルチャンネルにコンテンツを提供します。Cloud Playoutチャネルの主な利点は、BrightcoveがEPGを生成することです。そのためにサードパーティのサービスは必要ありません。

基礎データセクション

Brightcove Beaconでチャネルを作成するには、次の手順に従います。

  1. Brightcove Beacon にログインします。
  2. 上部のナビゲーションで、チャンネル。
  3. クリック新しいチャンネルを追加するボタン。
  4. の中に基本データタブ、定義名前あなたのチャンネルのために。
  5. Beacon チャネルのソースとしてCloudPlayoutチャネルを使用している場合は、CloudPlayoutチャネルIDを入力します。

    Cloud PlayoutチャネルIDを見つけるには、を参照してください。 Beacon チャネルの情報を取得する。

  6. の中にコンテンツのレーティングドロップダウンメニューで、チャンネルがストリーミングされる国のコンテンツレーティングシステムに基づいて、チャンネルのコンテンツレーティングを選択します。例の画像では、米国向けですTV-Y。
  7. の中にジャンル詳細セクションをクリックし、ジャンルチャンネルのジャンルを選択するためのドロップダウンメニュー。(例:ホラー、ドラマなど）
  8. チャンネルに適したジャンルが見つからない場合は、[]をクリックして追加のジャンルを作成できます。ジャンルを作成するボタン。
    9. NS サブジャンルドロップダウンメニューは、現時点では機能していません。

テキストデータ

ザ・テキストデータセクションでは、チャンネルに表示可能なテキストを追加できます。

  1. [ テキストデータ ] タブを選択します。
  2. の中に題名フィールドで、チャンネルのタイトルを定義します。
  3. の中に説明フィールドに、チャンネルの説明を定義します。
  4. の中にあらすじフィールドに、チャンネルの概要を定義します。
    5. あなたは追加することができますテキストデータチャンネルで使用するすべての異なる言語のセクション。

テキスト以外のデータ

ザ・非テキストデータセクションでは、チャンネルのポートレート画像とランドスケープ画像を追加できます。ポートレートとランドスケープの画像を提供します。画像が次の要件を満たしていることを確認してください。
イメージ 要件
ポートレートポスター ポスター画像 (2:3)
最小構成 幅:1000px 高さ:563px
[最大] 幅:4000px 高さ:2250px
風景ポスター 風景画像 (16:9):
最小構成 幅:1000px 高さ:563px
[最大] 幅:4000px 高さ:2250px

  1. を選択非テキストデータタブ。
  2. 追加する肖像画のポスターボックスセクションに画像をドラッグアンドドロップして、チャンネルの画像を表示します。
  3. 追加する風景ポスターボックスセクションに画像をドラッグアンドドロップして、チャンネルの画像を表示します。
目的の言語タブを選択するだけで、チャンネルで使用する言語ごとに異なるポートレートとランドスケープの画像セットを追加できます。

ストリーム

ザ・ストリームセクションでは、再生URLのオプションと権限を設定できます。

  1. を選択ストリームタブ。
  2. プロバイダを選択します。ほとんどの場合、Brightcove Video Cloud アカウントを参照するプロバイダーは 1 つだけです。
  3. [ 新しいストリームを追加 ] ボタンをクリックします。
  4. の中にストリーミングビデオIDフィールドに、VideoCloudライブイベントからビデオIDを貼り付けます。 Beacon チャネルのソースとしてクラウドプレイアウトチャネルを使用している場合は、を参照してください。 Beacon チャネルの情報を取得します。
  5. の中に公開するデバイスセクションで、公開するデバイスを追加します。[ Select/Unselect All Devices ] を選択して、使用可能なデバイスのリストを表示します。
  6. の中にストリームタイプ選択する住む。
  7. [ DRM タイプ] と [ストリーム形式 ] の値を変更する必要はありません。

権利とスケジュール

[ 権利とスケジュール ] セクションでは、公開ステータスと可用性を設定します。

  1. [ 権限とスケジュール ] タブを選択します。

  2. 必ず状態に設定されています公開あなたのチャンネルのために。
  3. 「可用性の詳細」については、次のガイドを使用して情報を入力します。
    フィールド 必須 注
    権利タイプ はい 収益化モデルに応じて選択します。
    • 自由
    • 広告-AVOD：広告で収益化されたチャンネルの場合。
    • サブスクリプション-SVOD：サブスクリプションで収益化されたチャンネルの場合。
    開始時刻 はい チャンネルの開始時間をスケジュールします。
    終了時刻 はい チャンネルの終了時間をスケジュールします。
    最大ストリーム数 該当なし 現時点では機能しません。
    公開するデバイス はい 公開するデバイスを追加します。[ Select/Select All Devices] を選択して、使用可能なデバイスのリストを表示します。
    許可された場所 はい コンテンツをストリーミングする場所を選択します。
    拒否された場所 いいえ コンテンツをストリーミングしない場所を選択します。

    地理的な場所を追加または編集するには、に移動します設定その後場所。
  4. 必ずクリックしてください新しいチャネルを作成する作業を保存するためのボタン。

チャンネルを電子番組ガイド（EPG）に追加する

チャンネルが作成されたら、最後のステップはそれをBrightcove BeaconEPGスケジュールに追加することです。これを行う方法については、次のWebサイトをご覧ください。電子番組ガイド（EPG）の作成ドキュメンテーション。

Beacon チャネルのソースとしてクラウドプレイアウトチャネルを使用している場合、BrightcoveがEPGを作成するため、これ以上のアクションは必要ありません。
