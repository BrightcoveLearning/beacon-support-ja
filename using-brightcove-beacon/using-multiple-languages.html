アプリの言語を選択してください

販売前のプロセスで、アプリを複数の言語に翻訳する意向を表明しました。翻訳の可能性には、アプリフィールドとコンテンツメタデータが含まれます。

アプリの翻訳は、アプリエクスペリエンスのアプリフィールドの翻訳をリクエストすることで実行されます。コンテンツの翻訳は、CMSに入力するメタデータです。タイトルや説明などのテキストデータや、テキストを含む画像が含まれます。

Brightcove Beaconアプリで使用する言語を定義することは、オンボーディングプロセスの一部です。オンボーディングプロセスを過ぎて、複数の言語の使用を開始することを選択した場合、または翻訳が存在する言語を追加したい場合は、Brightcoveのサポート担当者に連絡してください。新しい言語の翻訳については、Brightcoveサポートにお問い合わせください。

何が翻訳されていますか？

翻訳される/翻訳できるアプリには2つの部分があります。それらは次のとおりです。

  • アプリのフィールド名、ボタン、説明の翻訳は、お客様が提供することも、Brightcoveがお客様に代わって翻訳することもできます。
  • ページ名やビデオメタデータなど、定義したアイテムの翻訳、および画像は、ユーザーがBrightcove Beacon に入力します。

視聴者が動画アセットの言語を選択する方法

アプリで使用できる言語に加えて、ビデオアセットで使用できる多言語のキャプションとオーディオトラックがある場合があります。ビデオに複数の言語が使用可能な場合、視聴者はビデオの再生中に目的の言語を選択します。再生中に歯車のアイコンを使用できます。そのアイコンをクリックするとダイアログが表示され、視聴者は次のように字幕やオーディオトラックの言語を選択できます。
