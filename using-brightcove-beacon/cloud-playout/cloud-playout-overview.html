はじめに

Brightcove Cloud Playoutは、VODカタログからのアイテムのスケジュールされたプレイアウトをライブフィードと組み合わせてプログラムし、BrightcoveLiveの信頼性とスケーラビリティを備えたリニアチャネルとしてストリーミングできるようにするVideoCloudStudioモジュールです。

次に、Cloud Playoutチャネルに関する情報を作成時に入力することにより、CloudPlayoutチャネルを Beacon チャネルとして使用できます。 Beacon チャンネル。Cloud Playoutチャネルを使用する主な利点の1つは、EPGサービスがCloud Playoutに組み込まれているため、サードパーティのEPGプロバイダーを見つけて統合する必要がないことです。

Cloud Playoutは、次のようなスケジュールされたプログラミングのチャネルに強力なエンドツーエンドのソリューションを提供します。

  • 24時間年中無休のリニアチャンネル
  • ポップアップ/仮想チャネル
  • シミュレートされたライブイベント

Cloud Playoutは、事前に録画されたコンテンツを活用した放送のようなライブTV体験を通じて、視聴者のエンゲージメントを高め、新しい収益機会を創出するのに役立ちます。既存のVODアセットを使用して、スケジュールされたライブチャネルを作成します。

特徴

CloudPlayoutの機能は次のとおりです。

  • Video CloudMediaライブラリからVODアセットをインポートします。
  • VODアセットとスケジュールされたライブフィードを切り替えます。
  • 簡単、迅速、かつ信頼性の高い方法で、直前のスケジュール変更にリアルタイムで対応します。
  • サーバーサイド広告挿入（SSAI）を使用してチャンネルコンテンツを収益化します。
  • プログラムをループするようにチャネルを構成します。
  • デバイスごとにプレイアウトを最適化します。
  • ドラッグアンドドロップで、実行状態であっても、いつでもプログラムスケジュールを並べ替えることができます。
  • チャネルのEPG（電子番組ガイド）を取得するためのAPI

ワークフローの概要

Cloud Playoutチャネルを作成し、それを Beacon アプリに追加するには、次の2つの部分があります。

  1. にビデオクラウド、チャネルを作成します。
  2. に Beacon 、CloudPlayoutチャネルをソースとして使用する新しい Beacon チャネルを作成します。

制約事項

以下の仮定と制限が適用されます。

  • 動画は、動的配信プロファイルを使用して取り込みまたは再トランスコードする必要があります。 MP4レンディションが必要です
  • CloudPlayoutでサポートされる最大入力ビットレートと解像度は20MBPS / 1080pです。これは、VODコンテンツとライブ入力の両方に適用されます。
  • Brightcoveは、ビデオ間の正しい移行と電子番組ガイドの精度を確保するために、少なくとも2分間のビデオを使用することをお勧めします。
  • 現在、ビデオに追加されたキャプション（テキストトラック）は、クラウドプレイアウトチャネルに表示されません。キャプションのサポートは、将来のリリースで追加される予定です。
  • ライブイベントの最小時間は5分です。
  • Cloud Playoutのすべてのライブイベントは、Video Cloudリモートアセットと1：1の関係を共有します。このリモートアセットを他のチャネルで再利用することはできません。ライブイベントがチャンネルから削除されると、関連するリモートアセットは自動的に削除されます。チャネルが削除されると、関連するリモートアセットも自動的に削除されます。
  • ビデオ共有現在、別のアカウントからアカウントに追加することは、CloudPlayoutチャネルに追加できません。
  • ライブイベントが現在再生されているが、アクティブなフィードが利用できない場合、ライブフィードが再開されるか、イベントが終了するまで、スレートコンテンツがループで再生されます。スレートMP4の再生が開始されるまでに約30秒の遅延があります。
  • Video Cloudアカウントが地理的制限用に構成されている場合、チャネル出力領域 VideoCloudの地理的制限ポリシーと互換性のあるリージョンで作成する必要があります。
  • 最高のエクスペリエンスを得るには、BrightcoveはGoogleChromeブラウザの使用をお勧めします。
  • SSAI対応チャネルの場合、宛先を広告対応RTPライブイベントに設定する必要があります。
