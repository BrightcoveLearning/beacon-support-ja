はじめに

Cloud Playoutチャネルの構成方法に応じて、さまざまな種類のコンテンツを追加できます。VODコンテンツとプレイリストはどのチャンネルにも追加できます。さらに、それをサポートするようにチャネルを構成した場合は、ライブイベントを追加できます。次のセクションでは、各種類のコンテンツを追加する手順について説明します。

あなたもできることに注意してくださいバンパーを追加するあなたのチャンネルに、そしてサーバーサイド広告、チャネルがそのように構成されている場合。その方法の詳細については、これらのトピックを参照してください。

VODコンテンツの追加

チャンネルを作成した後、またはチャンネルの詳細に移動すると、画面にチャンネル設定が表示されます。

詳細は編集できますが、次の主なステップはチャンネルにコンテンツを追加することです。

ビデオの追加（VOD）手順

  1. クリック ビデオを追加（チャンネルリストページを表示している場合は、チャンネル名をクリックしてチャンネル設定ページを開き、[ ビデオを追加）。
  2. チャンネルに追加する動画の横にあるチェックボックスをクリックします。
  3. クリックビデオを追加）
  4. クリック 保存する。
  5. チャネルのプログラムリストが表示されます。

    開始時間と終了時間は、ビデオの長さから自動的に計算されることに注意してください。リスト内のビデオをドラッグアンドドロップして並べ替えることができ、開始/終了時間が自動的に調整されます。

プレイリストを追加する

再生リスト全体をチャンネルにインポートすることもできます。次の条件に注意してください。

  • EXPLICIT（手動）プレイリストとスマートプレイリストの両方をインポートできます。
  • NS スナップショット現在の状態のプレイリストのがインポートされます（つまり、プレイリストへの将来の変更はいいえチャネルコンテンツに反映されます）。
  • 一度にインポートできるプレイリストは1つだけです。
  • インポートされたプレイリストにライブイベントが含まれている場合、同じ名前、説明、期間の同じ新しいライブイベントが作成されます。これは、ライブイベントが常にチャネルと1：1の関係を共有するためです。
  • 再生リスト内の動画が追加されます最後まで現在のチャンネルプログラムの。
  • プレイリストだけでなくVODコンテンツも追加でき、複製は許可されます。
  • 総資産制限（100）の検証チェックがあります。インポートが100アセット制限を超える場合、インポートは失敗します。
  • 注:チャンネルでライブ入力が有効になっていない場合でも、ライブイベントをインポートから除外することを選択することで、ライブイベントを含むプレイリストをインポートできます。
  • 注:これがSSAI対応チャネルの場合、ライブイベントは許可されません。ライブイベントをインポートから除外することを選択することにより、ライブイベントを含むプレイリストをインポートすることができます。
  • 注:MP4レンディションの対象となるアセットのみが許可されます。警告は、どのアセットがインポートに不適格になるかを示します。確認すると、適格なアセットのみがインポートされます。

プレイリストの手順を追加する

  1. クリック プレイリストをインポートする。
  2. クリック輸入インポートするプレイリスト用。
  3. プレイリストの一部のアイテムをインポートできないことを示すメッセージが表示された場合は、続行サポートされているビデオをインポートする場合。
  4. クリック 保存する。
  5. インポートが成功した場合は、プログラムリストに表示されます。

ライブイベントを追加する

注：これは、チャネルでライブイベントが有効になっている場合にのみ適用されます。

  1. クリック
  2. [ライブイベントの追加]ダイアログで、名前、デュレーション、および説明ライブイベントの場合：

    ライブイベントの最小期間は5分であり、期間は5分間隔でのみインクリメントまたはデクリメントできることに注意してください。必要に応じて、テキストボックスで期間をインラインで編集して、時間の粒度を調整できます。ライブイベントの期間は、現在再生中であってもいつでも変更できます。

  3. ライブイベントを追加すると、VODアイテムとともにプログラムリストに表示されます。
  4. ビデオと同じように、ライブイベントをドラッグしてリストを並べ替えることができます。また、使用することができます ライブイベントの期間を変更するボタン。

こちらもご覧くださいコンテンツへのバンパーの追加 VODまたはライブイベントアイテムにバンパーを追加するため。
