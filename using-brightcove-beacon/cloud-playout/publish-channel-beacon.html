はじめに

Cloud Playoutチャネルを作成したら、それを Beacon アプリの一部にするために実行する必要がある3つのステップがあります。これらについては、次のセクションで詳しく説明します。

Beacon チャネルの情報を取得する

先に進む前に、 Beacon チャネルとプレイリストの設定に必要な情報を取得する必要があります。

  1. チャンネルリストのチャンネルをクリックして、チャンネルの詳細ページに移動します。
  2. まだ行っていない場合は、をクリックします 活性化チャネルをアクティブにします。見るチャネルアクションの使用詳細については。
  3. チャネルがアクティブ化されたら、メインナビゲーションを使用してビデオクラウドに移動しますメディアモジュール。
  4. CloudPlayoutチャンネルと同じ名前の動画が見つかるはずです。画像のないライブビデオになります：
  5. ビデオ名をクリックして詳細を開きます。
  6. 両方を保存しますビデオIDそしてそのチャネルID（続く長い英数字の文字列クラウドプレイアウト：タグ内）。

テキストデータ

ザ・テキストデータセクションでは、チャンネルに表示可能なテキストを追加できます。

  1. [ テキストデータ ] タブを選択します。
  2. の中に題名フィールドで、チャンネルのタイトルを定義します。
  3. の中に説明フィールドに、チャンネルの説明を定義します。
  4. の中にあらすじフィールドに、チャンネルの概要を定義します。

ストリーム

[ストリーム]セクションで、チャンネルのストリーミング情報を設定します

  1. [ストリーム]セクションで、[ ブライトコーブとしてプロバイダーをクリックします新しいストリームを追加する：
  2. Steams情報を追加します。
    • ストリーミングビデオID：チャンネルのライブストリームの動画IDを入力します
    • 公開するデバイス： [すべてのデバイスを選択/選択解除]をクリックして、公開したくないデバイスをすべて削除します
    • ストリームタイプ：選択する住む
    • DRMタイプ：使用しているDRMタイプを選択するか、開ける
    • ストリーム形式： HLSを選択します

権利とスケジュール

  1. の中に権利とスケジューリングセクションに、次の情報を追加します。
    • 状態： [公開済み]を選択します
    • ダウンロードを許可する：視聴者にコンテンツのダウンロードを許可するかどうかを選択します（ほとんどの場合、答えはノーになります）
    • 権利の種類：無料またはコンテンツに適したものを選択してください
    • 始まる時間：チャンネルを利用できるようにする時間（過去の日付を選択してすぐに利用できるようにします）
    • 終了時間：チャネルが使用できなくなる時間（無期限に実行したい場合は、将来的に設定できます）
    • 最大ストリーム:何も入力する必要はありません-0のままにしておきます
    • 公開するデバイス： [すべてのデバイスを選択/選択解除]をクリックして、公開したくないデバイスをすべて削除します
    • 許可された場所：入力世界または表示を許可する領域
    • 拒否された場所：空白のままにするか、表示が許可されていない領域に入る

チャンネルを再生リストに追加する

Beacon ページはプレイリストコンテンツの場合、CloudPlayoutチャンネルをプレイリストに追加する必要があります。再生リストには他の動画を含めることができますが、通常はチャンネルのみが含まれるため、ページのコンテンツで構成されます。見るプレイリストの作成詳細については。

Beacon ページにチャンネルを追加する

Cloud Playoutチャンネルを Beacon アプリに配置する最後のステップは、最も簡単なステップです。必要なのは、チャンネルビデオを含むプレイリストをページに追加することだけです。見るページの作成詳細については。
