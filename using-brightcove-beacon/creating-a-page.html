新しいページの作成

このトピックでは、新しいBrightcove Beacon ページを作成する方法を学習します。

概要

Brightcoveの Beacon ページでは、ユーザーが動画コンテンツを選択して視聴することができます。カルーセルには、Video Cloud Studio で定義した動画または再生リストのグループが表示されます。Brightcove Beacon管理コンソールを使用すると、コンテンツカルーセルを含む新しいページを作成できます。

ページの作成

新しいページを作成するには、次の手順に従います。

  1. Brightcove Beacon にログインします。

  2. 上部のナビゲーションで、[ ページ ] を選択します。

  3. を選択新しいページを追加ボタンをクリックして、新しいページを作成します。

  4. のために基本データタブで、次のフィールド値を入力します。

    • システムタイトル -内部システムページ名。これはユーザーには表示されませんが、わかりやすいURLを生成するために使用されます。後の手順で、表示可能なタイトルを入力します。
    • 注文 -ホームページでのページ注文
    • ジャンルタイプ -値：
      • 主人 - メインページ
      • 子 -従属ページ
    • ナビゲーションタイプ -ページテンプレートのリストから選択します
    • プレゼンテーションタイプ -値：
      • icontext -テキストを含むアイコンを表示します
      • アイコン -アイコンを表示する
      • テキスト -テキストを表示
    • 画面レイアウト -値:(すべてのナビゲーションタイプに画面レイアウトがあるわけではありません）
      • タイトルの画面レイアウト -タイトル付きのレイアウト
      • タイトルのない画面レイアウト -タイトルのないレイアウト
    • 家ですか？ -これがホームページかどうかを選択します
  5. 次に、ページにブロックを追加します。ブロックはプレイリストのコンテナです。サインインタイプやログアウトタイプなど、一部のナビゲーションタイプにはブロックがありません。を選択しましたナビゲーションタイプの値で画面。

  6. [ ブロックを追加 ] ボタンを選択します。ボタンを選択すると、右側にブロックタイプが表示されます。

  7. 必要なブロックの種類を選択します。この場合、リストブロックを選択しています。目的のブロックに関連付けられた [ブロックを追加 ] ボタンを選択します。

  8. [ レイアウトの追加 ] ボタンを選択します。ボタンを選択すると、レイアウトタイプが右側に表示されます。

  9. [注目]、[カルーセル]、[グリッド] のいずれかで、必要なレイアウトの種類を選択します。この例では、レイアウト内のカルーセル風景ラベルを選択しています。

  10. を選択プレイリストを追加ボタン。ボタンを選択すると、プレイリストの種類が右側に表示されます。

  11. プレイリストを選択します。この例では、ユーモアのプレイリストに関連付けられた [ プレイリストの追加 ] ボタンを選択しています。

    Video Cloud Studioを使用してプレイリストを作成し、VideoCloudアカウントに同期するとプレイリストがここに表示されます。

  12. ユーモアプレイリストがカルーセルに追加されていることがわかります。ドラッグアンドドロップして、アイテムを並べ替えることができます。

  13. のためにテキストデータ、非テキストデータ、タグそして権利とスケジューリングタブ、を参照してください共通の構成設定資料。

  14. を選択新しいページを作成するボタンをクリックして、新しいページを作成します。あなたはすべて設定されています。新しいカルーセルが Brightcove Beacon ページに表示されます。
