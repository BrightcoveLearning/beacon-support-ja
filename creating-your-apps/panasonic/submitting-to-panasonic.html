はじめに

Brightcove Beacon を使用すると、Panasonic Store に追加できる生成されたデバイスアプリが届きます。

  • 店舗詳細: https://developer.vieraconnect.com/
  • 承認のための推奨リードタイム:三週間

パナソニックに必要な情報:

  • アプリケーション
    • アプリケーション URL とアプリケーションのステージング URL
  • 説明
    • 最大1024文字
    • 最大 26 行
    • 最大85文字（またはアジア文字44文字）で構成された各行
    • 適切な場所に改行を挿入してください
  • アイコン
    • 4 つのアイコンイメージが必要です。
      • 160x112 (PNG)
      • バナースタイルのアイコン 256x60 (PNG)
      • 360x252 (PNG)
      • 円形のアイコンの直径 336x336 (PNG)
  • スクリーンショット
    • 少なくとも1つ必要です
    • 480w × 270H PNG
    • これらは、マーケットのアプリケーション詳細ページに表示されます

入門編

Panasonic の開発者ページにアクセスするには、次の手順を実行します。

  1. ブラウザで、 Panasonic 開発者ログインページに移動します。

  2. アカウントでサインインするか、新しいアカウントを作成します。

  3. ログインすると、コントロールパネルが表示されます。

  4. [ 新しい申請を送信 ] ボタンの右側にあるメモを読みます。ホームページのリンクを選択します。

  5. アプリの要件を確認します。ご心配なく、Brightcoveは必要なテクノロジーでアプリを作成します。

アプリケーション情報

[ アプリケーション情報 ] セクションには、アプリの基本情報を入力します。

  1. コントロールパネルから、[ 新しいアプリケーションの送信 ] ボタンを選択します。

  2. 次のような入力フォームが表示されます。

  3. 「アプリケーション情報」ページで、次の必須フィールドに値を入力します。

    • プロジェクトの種類 -ドロップダウンメニューから、利用可能な最新の HTML バージョンを選択します。
    • アプリケーション URL -Brightcove からアプリビルドをホストする 2 つのリポジトリを作成します。1 つはプロダクション用、もう 1 つはステージング用です。アプリケーション URL は本番 URL です。
    • ステージング URL -アプリビルドのステージング URL を入力します。
    • バージョン番号 -アプリのバージョンを入力します。

  4. [ 追加/削除 ] ボタンを選択します。アプリで使用したい言語を選択します。[ 確認 ] ボタンを選択します。

  5. 選択した言語でフィールドの値を入力します。この例では、英語 (北米) が選択されています。

    • デフォルトの言語 -ドロップダウンメニューから、アプリの初期言語を選択します。アプリが英語をサポートしている場合、Panasonic では、英語（北米）または英語（英国モデル）を選択することをお勧めします。
    • アプリケーション名 -アプリに使用する名前を入力します。
    • 短縮名 -アプリに使用する短縮名を入力します。これは、アプリケーション名と同じにすることができます。
    • キーワード -アプリを検索するためにマーケット検索機能で使用するキーワードを入力します。キーワードはコンマで区切ります (例:卓球、スポーツ)。最大 128 文字です。
    • 説明 -アプリの説明を入力します。これは、マーケットのアプリケーション詳細ページに表示されます。詳細については、入力ボックスの内側をクリックします。
    • エンドユーザー使用許諾契約 -オプション。エンドユーザー使用許諾契約を入力します。

  6. 選択した言語のリストで次の言語を選択し、前の手順で説明したフィールドに値を入力します。

  7. [ 次へ ] ボタンを選択します。

その他の詳細

[ 追加の詳細 ] セクションでは、サポートおよびコンテンツの詳細を追加できます。

  1. 前のセクションから [ 次へ ] ボタンを選択すると、次のような入力フォームが表示されます。

  2. [ その他の詳細 ] ページで、必須フィールドの値を入力します。

    • サポートメール -アプリに問題が発生したときにユーザーが連絡する電子メールアドレスを指定します。
    • ペアレンタルレーティング -ドロップダウンメニューを展開して、各ペアレンタルレーティングの定義を確認します。アプリに適した評価を選択します。
    • ブラジルのコンテンツ評価 -ブラジルでリリースを計画している場合は、必須です。ドロップダウンメニューを展開して、ブラジルの各レーティングの定義を確認します。適切な評価を選択します。
    • 連絡先の電子メール -連絡先のメールアドレスを入力します。
    • Category : ドロップダウンメニューを展開して、カテゴリのリストを表示します。適切なカテゴリを選択します。
    • 価格 -通常、パナソニックのアプリは無料でダウンロードできます。
    • アプリ内購入 -アプリ内購入をアプリにするかどうかを選択します。
    • マーケットプレイス -国のリストを表示するには、 [マーケットプレイスの選択 ] ボタンをクリックします。[ すべての国 ] を選択するか、アプリを利用できるようにする国を選択します。完了したら、[完了] ボタンを選択します。
    • QAの手順 -ボタンの追加、リンクの変更など、役立つ指示を記載してください。必要に応じて、アプリのテストアカウントを含めます。

  3. [ 次へ ] ボタンを選択します。

アセットのアップロード

「アセットのアップロード」セクションでは、アイコンや画像を追加できます。

  1. 前のセクションから [ 次へ ] ボタンを選択すると、次のような入力フォームが表示されます。

  2. 「アセットのアップロード」ページで、必須フィールドに値を入力します。

    • アプリ説明ドキュメント -アプリの動作の詳細が記載されたドキュメントを作成します。これは、LG用に作成したドキュメントと同じにすることができます。[ 参照 ] ボタンを選択して、ファイルを検索します。次に、[ アップロード ] ボタンを選択します。

      アップロードできるのはdoc、docxpdfまたはファイルのみです。最大ファイルサイズは 8MB です。

    • アイコン1-マーケットアイコンをアップロードします。このアイコンは、マーケットの「リストページ」と「アプリケーション詳細ページ」に表示されます。

      160W × 112H PNG

    • アイコン 2 (円形) -Firefox OS のアイコンをアップロードします。このアイコンは、マーケットの [ アプリドキュメント] ページに表示されます。

      336 ピクセルの直径の円形 PNG

    • サンプル画像 1 -これはアプリのスクリーンショットで、マーケットのアプリケーション詳細ページに表示されます。サンプル画像を 5 つまで追加できます。

      480w × 270H PNG

  3. [ 次へ ] ボタンを選択します。

契約に同意する

[ 同意する ] セクションでは、ストアに送信する前に、アプリに追加したすべての情報を確認できます。パナソニック利用規約のセクションもあります。

  1. 前のセクションから [ 次へ ] ボタンを選択すると、次のような入力フォームが表示されます。

  2. アプリに追加した情報と画像を確認します。

  3. パナソニックテレビの利用規約をお読みください。読んだり同意した旨のボックスにチェックを入れます。

  4. 「発行」ボタンを選択します。

  5. [ アプリケーションのリスト ] ページで、アプリが下書きとして保存されているはずです。

  6. アプリはライブになる前に QA ステータスになります。
