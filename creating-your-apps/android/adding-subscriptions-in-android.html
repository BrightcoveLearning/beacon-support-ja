はじめに

このドキュメントでは、次の 2 つのトピックに焦点を当てています。

  • サブスクリプションの作成
  • サービスアカウントの作成

サブスクリプションは、Brightcove Beaconで作成してから、 グーグルプレイコンソール。Brightcove Beacon でサブスクリプションを作成する手順については、「サブスクリプションパッケージの作成」ドキュメントを参照してください。このドキュメントでは、Android用の対応するサブスクリプションを作成する方法について説明します。

サービスアカウントを使用すると、Google Play ストアで Brightcove Beacon アプリを更新できます。これは、サブスクリプションをテストする場合に特に役立ちます。この文書の後半では、サービスアカウントを作成し、Google Play サーバーからサーバーへのサブスクリプションに必要なファイルを生成する手順について説明します。

サブスクリプションの追加

Android でサブスクリプションを作成するには、Brightcove Beacon で作成したサブスクリプションに関する情報が必要です。サブスクリプションを作成するには、次の手順に従います。

  1. Google Play コンソールにログインし、アプリをクリックします。
  2. Google Playコンソールの [ サブスクリプション ] セクションに移動する必要があります。このプロセスは、サブスクリプションを以前に作成したかどうかによって若干異なります。選ぶ 一 以前のサブスクリプションの作成に応じて、このステップのサブセクションの

    以前のサブスクリプションはありません

    1. 左側のナビゲーションから [ > アプリ内製品のサブスクリプション ] を選択すると、次のページが表示されます。
    2. 出品者アカウントの設定ボタンをクリックします。
    3. 出品者アカウントの設定リンクをクリックします。
    4. フォームに入力して [ 送信 ] をクリックすると、新しいサブスクリプションを追加できるサブスクリプションページが表示されます。

    サブスクリプションが存在します

    1. 左側のナビゲーションから [ > アプリ内製品のサブスクリプション ] を選択すると、次のページが表示されます。
    2. 以前に作成したサブスクリプションが表示されることに注意してください。
  3. [ サブスクリプションの作成 ] ボタンをクリックします。
  4. フォームで、まず製品 ID を入力します。この値は、Brightcove Beacon でサブスクリプションを作成するときに指定した名前と一致する必要があります。
  5. デフォルトの言語を入力します。
  6. サブスクリプションの詳細を入力します。必須ではありませんが、[ 名前] と [ 説明 ] を Brightcove Beacon のサブスクリプション定義と一致させることがベストプラクティスです。
  7. 請求期間とデフォルト価格を入力します。これらの値は、Brightcove Beacon で作成されたサブスクリプションと一致する必要があります。値が一致しない場合は、Brightcove Beacon アプリでは 1 つの価格が表示されますが、Google Play ストアでは異なる料金が課金されます。
  8. サブスクリプションオプションの値を入力します。必須ではありませんが、Brightcove Beacon のサブスクリプション定義に対応する値を一致させることがベストプラクティスです。
  9. [ 保存 ] ボタンをクリックして、サブスクリプションを作成します。

サービスアカウントの作成

Google Play サーバーアカウントを作成するには、次の手順に従います。

  1. Google Play コンソールにログインします。

  2. 左側のナビゲーションで、[ 設定 ] を選択します。

  3. 左側のナビゲーションの [ 開発者アカウント ] で、[ API アクセス ] を選択します。

  4. [ +サービスアカウントの作成 ] ボタンを選択します。

  5. [ サービスアカウントの作成 ] ダイアログで、手順 1 の [ Google API コンソール ] リンクを選択します。

  6. Google APIコンソールが表示されます。

  7. 左側のナビゲーションでは、サービスアカウントがすでに選択されています。青いヘッダーバーの下にある [ サービスアカウントの作成 ] ボタンを選択します。

  8. サービスアカウントの名前を入力します。サービスアカウント ID が自動的に入力されます。説明を追加します。[ 作成 ] ボタンを選択します。

  9. [ サービスアカウントの権限 ] 画面で、[ ロールの選択] ドロップダウンメニューを展開し、 [ 所有者 ] を選択します。

  10. [ 続行 ] を選択します。

  11. 次に、Brightcoveで使用できるキーを作成します。[ +キーを作成 ] ボタンを選択します。

  12. サイドパネルで、デフォルトのキータイプを JSON のままにします。[ 作成 ] ボタンを選択します。

  13. 秘密鍵ファイルをコンピュータにダウンロードし、[ 閉じる ] ボタンを選択します。

  14. ダウンロードした秘密キーファイルをBrightcoveに送信します。
  15. [ 完了 ] を選択します。

  16. 独自の値で次の画面が表示されるはずです。このブラウザタブを閉じて、Google Playコンソールに戻ります。

  17. Google Play コンソールで、[ アクセスを許可 ] ボタンを選択します。

  18. [ 新しいユーザーの追加 ] ダイアログで、[ 財務データ ] ボタンがオンになっていることを確認します。[ ユーザーの追加 ] ボタンを選択します。

Google Play サービスアカウントが正常に作成されました。
