はじめに

Brightcove Beacon では、生成されたデバイスアプリを受け取り、Roku チャンネルストアに追加することができます。

  • 店舗詳細: https://developer.roku.com/overview
  • 開発者アカウントのセットアップ: https://my.roku.com/signup
  • ロク検索の実装: https://developer.roku.com/en-gb/docs/developer-program/discovery/search/implementing-search.md
  • 承認のための推奨リードタイム:三週間
チャンネルがすべてのデバイスで検索可能になるまで、公開後最大48時間かかる場合があることに注意してください。Rokuサイトから：チャンネルが公開されたばかりですが、検索で見つからないのはなぜですか？

Rokuに必要な情報:

  • ポスターとスクリーンショットの要件
    • チャンネルポスター (540x405)
    • 1920x1080までのスクリーンショット
  • 開発者Eメール
  • 詳細については推奨 URL
  • 管理連絡先 (氏名、電話番号、電子メール)
  • 技術連絡先 (氏名、電話番号、電子メール)
  • ストアで QA 用のアプリログインをセットアップするクライアント

入門編

Roku の開発者ページにアクセスするには、次の手順を実行します。

  1. ブラウザで、 Roku サインインページに移動します。

  2. アカウントでサインインするか、新しいアカウントを作成します。

チャンネルの追加

チャンネルを追加する方法について説明します。

  1. 上部のナビゲーションで、プロファイルアイコンをクリックします。

  2. ドロップダウンメニューで、開発者オプションをクリックします。

  3. 上部のナビゲーションから、[ ダッシュボード ] リンクを選択します。

  4. ダッシュボードから、チャンネルを管理できます。[ チャンネルを追加 ] ボタンを選択します。

  5. Developer SDK オプションを選択すると、パブリックチャネルまたは非認定チャネルのオプションが表示されます。

  6. チャンネルをストアに移動する場合は、[ 公開 ] を選択します。テストチャネルを作成する場合は、[ 非認定 ] を選択します。

    チャネル名は両方で一意である必要があります公衆と認定されていませんチャネル。

    次のことを推奨します。

    • 公開チャンネル名 (例: 送信先 ) を作成します。

    • を作成します認定されていません接尾辞がQA。したがって、この場合、それは宛先 QA という名前になります。

    まず、パブリックチャンネルを作成しています。

  7. 次に、テストチャンネルを作成しています。

チャンネルのプロパティ

プロパティは各チャンネルに関連付けられます。

  1. 開発者ダッシュボードに戻り、[ マイチャンネルの管理 ] リンクを選択します。

  2. チャンネルの [ プレビューと更新 ] ボタンを選択します。

  3. [ プレビューと公開 ] ドロップダウンメニューを展開し、[ プロパティ ] を選択します。

  4. チャンネルのプロパティのリストが表示されます。左側には、チャンネルが表示されるチャンネルストアの国の一覧が表示されます。テストの場合は、すべての国を選択できます。ライブチャンネルで、チャンネルを表示する国を選択します。

  5. アクセシビリティ機能とコンプライアンスについては、[ はい ] を選択します。Brightcove Beacon アプリはすべて、これらの要件を満たしています。

  6. コンテンツに基づいて、適切なコンテンツレーティングを選択します。

  7. チャンネル機能にアクセスするためにカスタマーアカウントが必要かどうかを選択します。

  8. アプリ全体に関連付けられているプライバシーポリシーの URL を入力します。

  9. バニティアクセスコードは Roku によって自動的に生成されます。この値は残しておくことができます。

  10. サポートしたい言語を選択します。

  11. [ 必須機能 ] で、まず、チャンネルに最も適した分類を選択します。

  12. 次に、アプリにインターネット接続が必要かどうかを指定します。チャンネルのプロパティの更新が完了したら、[ 保存 ] ボタンを選択します。

チャンネルストア情報

ここでは、チャンネルの説明とアイコンを追加します。

  1. [プレビューと公開] ページに戻り、[ プレビューと公開 ] ドロップダウンメニューを展開し、[ チャンネルストア情報 ] を選択します。

  2. [チャネルストア情報] で、次のフィールド値を入力します。

    • チャンネル名 -チャンネル名が既に入力されているはずです。
    • 説明 -チャンネルについての詳細な説明を追加します。この説明は、チャンネルストアの説明に表示されます。（300文字）
    • ウェブの説明 -米国のウェブベースのマーケティングチャンネルの説明を入力します。これはデバイスには表示されません。(1500文字)
    • チャンネルポスター -Rokuチャンネルストアに表示されるポスター画像を追加します。（540×405 JPEGまたはPNG）
    • 説明カテゴリ -チャンネルを表示するカテゴリを選択します。(例:スポーツ、ドラマ、コメディなど）
    • 国内地域 -チャンネルが母国または国内とみなされる国を選択します。チャンネルは、国内地域に関連付けられているチャンネルストアを除き、チャンネルを含むすべての地域のチャンネルストアの「国際」カテゴリに表示されます。

  3. チャンネルストア情報の追加が完了したら、[ 保存 ] ボタンを選択します。

アカウントを確認し、デバイスをリンクします

Rokuチャネルの設定が完了したら、開発者アカウントを確認することが重要です。

  1. に戻るチャネルの管理画面を選択し、プレビューと更新ボタン。

  2. 展開プレビューと公開選択しますパッケージのアップロード。

  3. 選択する確認メールを送信ボタン。

    チャネル設定の開始時に、開発者の権利に使用される組織の電子メールアドレスを使用していることを確認してください。

  4. 開発者の電子メールアカウントを確認すると、デバイスをリンクして支払い方法を追加するように求められます。すでにこれを行っている場合は、次のセクションにスキップできます。

  5. Rokuデバイスをリンクするときは、デバイスに確認手順と同じ登録済みメールアドレスがあることを確認してください。最良の結果を得るには、Rokuデバイスをリセットすることをお勧めします。

    Rokuデバイスをリセットするには：

    1. 選択する設定。
    2. 選択するシステム。
    3. 選択する高度なシステム設定。
    4. 選択する工場出荷時のリセット。

    ここに役立つリンクがあります：

    • RokuTVをリンクする
    • Roku Express / Express +の設定方法

  6. 完了すると、次の画面が表示されます。

コードでチャネルを追加する

バニティコードを使用して、アプリチャネルをRokuデバイスに追加できます。

  1. プロファイルのドロップダウンメニューを展開して、マイアカウント。

  2. を選択コードでチャネルを追加するリンク。

  3. の中にチャネルを追加ダイアログで、次の手順を実行します。

    1. 入力しますチャネルアクセスコード私たちがあなたに提供するもの。
    2. を確認してください私はロボットではありません箱。
    3. 選択するチャネルを追加。

  4. チャンネルがRokuアカウントに追加されたら、次の手順に従ってRokuTVでアプリを確認します。

    1. 選択する設定。
    2. 選択するシステム。
    3. 選択するシステムアップデート。

    これで、レビューのためにアプリが表示されます。

収益化

ここでは、チャンネルに関する収益化情報を追加します。

  1. [プレビューと公開] ページに戻り、[ プレビューと公開 ] ドロップダウンメニューを展開し、[ 収益化 ] を選択します。

  2. チャンネルに追加を含めるか、サブスクリプションモデルをサポートするかを指定します。

    Brightcove SSAIを使用している場合でも、Roku'sに登録する必要があることに注意してください。パートナーペイアウトプログラム。広告の再生中にAdsコールバックを行うことはRokuストアの要件です。

    サブスクリプションチャンネルの場合は、次の情報が含まれます。

    • 購入タイプ -チャネルに対するユーザーの請求方法。（一回、毎月、毎年など）
    • 価格帯 -ドロップダウンメニューから値を選択します。EU チャネルストアの国の購入価格には VAT が含まれています。収益は税引前（正味価格）に基づきます。

  3. 収益受け取り情報の追加が完了したら、[ 保存 ] ボタンを選択します。

スクリーンショット

ここでチャンネルにスクリーンショットを追加できます。

  1. [プレビューと公開] ページに戻り、[ プレビューと公開 ] ドロップダウンメニューを展開し、[ スクリーンショット ] を選択します。

  2. チャンネルストアに表示されるスクリーンショットを追加します。

    スクリーンショットは最大 6 枚まで追加できます。それぞれが 1920 x 1080 の JPEG または PNG ファイルである必要があります。

  3. スクリーンショットの追加が完了したら、[ 保存 ] ボタンを選択します。

サポート情報

チャンネルのサポートに役立つ連絡先情報を提供します。

  1. [プレビューと発行] ページに戻り、[ プレビューと発行 ] ドロップダウンメニューを展開し、[ サポート情報 ] を選択します。

  2. [サポート情報] で、次のフィールド値を入力します。

    • 開発者用メール -開発者のメールアドレスを入力します。正しくない場合は、連絡先情報を更新してください。
    • [ 優先 URL ]-チャンネルの URL を指定します。これは、Rokuのウェブサイトやニュースレターがチャンネルにリンクするために使用されることがあります。
    • エンドユーザサポート連絡先 -Web ページ、電子メールアドレス、電話番号、またはそれらの複数を入力します。Roku は、チャンネルについてカスタマーサービスに連絡した場合に、これを使用します。（300文字）これは必須です。

    • テストアカウントとディープリンクパラメータ -ディープリンクとして開きたいアセットをBrightcoveに伝え、ディープリンクのパラメータを更新します。アカウント共有を設定したくない場合は、Brightcove からパラメータが提供されます。

      また、3 つのテストアカウントの資格情報も提供します。これらのアカウントは、カスタマーサポート、デモ、QA に使用されます。（1500 文字）異なるサブスクリプションのアカウントをお持ちの場合は、チャンネルストアがすべてのアカウントにアクセスできるように、ここにリストする必要があります。

    • 管理連絡先 -アプリの管理連絡先情報を入力します。
    • 技術連絡先 -アプリの技術連絡先情報を入力します。

  3. サポート情報の追加が完了したら、[ 保存 ] ボタンを選択します。

パッケージのアップロード

パッケージをチャンネルにアップロードします。

  1. [プレビューと公開] ページに戻り、[ プレビューと公開 ] ドロップダウンメニューを展開し、[ パッケージアップロード ] を選択します。

  2. まず、メールアドレスを確認する必要があります。その後、アップロードできるようになります。

  3. 「パッケージのアップロード」で、次のフィールド値を入力します。

    • チャンネルバージョン -バージョン 1.0 から、アプリのバージョンを指定します。
    • 最小ファームウェア -Roku ファームウェアのバージョンを選択します。パッケージは、最低限のファームウェアバージョン以上をサポートするデバイスでのみ実行されます。推奨される最新の安定版ビルドバージョンは Roku v9.2 bl です。
    • パッケージファイル -パッケージファイルは、お客様またはBrightcoveがアップロードできます。このファイルの名前は、次のようになります。Beacon_1.0.6138.pkg。

  4. パッケージのアップロード情報の追加が完了したら、[ 保存 ] ボタンを選択します。

静的解析

チャンネルに関する分析レポートを作成します。

  1. [プレビューとパブリッシュ] ページに戻り、[ プレビューとパブリッシュ ] ドロップダウンメニューを展開し、[ 静解析 ] を選択します。

  2. 前のセクションでパッケージをアップロードすると、パッケージの構造または構文のエラーが分析されます。テーブルには、エラーと警告が表示されます。

プレビューと公開

チャンネルに関連するフォームをすべて入力すると、 [プレビューと公開 ] 画面が表示されます。

  1. [プレビューと公開] ページに戻り、[ プレビューと発行 ] ドロップダウンメニューを展開し、[ プレビューと発行 ] を選択します。

  2. このページでは、チャンネルの概要プレビューが表示されます。

  3. 準備ができたら、[ 送信 ] ボタンを選択してチャンネルを公開します。

  4. アプリは Roku ストアによってチェックされ、問題がないかどうかが確認されます。何らかの理由でチャンネルが拒否された場合、[ サポート情報 ] 画面で設定した開発者メールにメールが送信されます。拒否理由を確認し、Brightcoveサポートチケットを作成して問題を解決してください。

チャンネルを更新する

既存のアプリを更新する場合は、次の手順に従います。

  1. 新しい Roku パッケージを作成します。

    新しいパッケージに一意の ID とパスワードを指定します。このパッケージのキーと共に、新しいパッケージを Brightcove に提供します。ストアに新しい荷物を提出するには、新しい情報が必要です。

    チャンネル名を変更する場合は、チャンネルストア内で一意である必要があります。
