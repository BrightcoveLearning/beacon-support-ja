はじめに

チャネル内商品とは、月間サブスクリプションや年間サブスクリプションなど、チャンネル内で提供できる収益化ソリューションです。このドキュメントでは、Roku アプリケーションチャネルを収益化するチャネル内製品の作成方法について説明します。

Rokuでのチャネル内製品の作成

以下の手順に従って、Roku Developers ページでチャネル内製品を追加する方法について説明します。

  1. 上部のナビゲーションで、プロファイルアイコンをクリックします。

  2. ドロップダウンメニューで、開発者オプションをクリックします。

  3. 上部のナビゲーションから、[ ダッシュボード ] リンクを選択します。

  4. [ チャネル内製品の管理] を選択します。

  5. 新しい商品登録ボタンをクリックします。
  6. [ Channels ] フィールドで、製品を販売できるチャネルを選択します。
  7. フォームに必要事項を記入します（手順のリストの下にある参照スクリーンショット）。
    1. [ 製品カテゴリ] で [ ビデオ ] を選択します。
    2. [ 製品名 ] に、顧客に表示される名前を入力します。別の言語で商品名を登録するには、[ 別の言語で商品名を追加 ] ボタンをクリックします。
    3. のために製品識別子、内部参照IDを入力します。
      内部参照IDは一意である必要があり、後でIn-Channel製品をBrightcove BeaconCMS構成に追加するために必要になることに注意してください。

    4. [ 購入タイプ ] で、チャネル内製品の購入者に請求される方法に基づいて、サブスクリプションの種類を選択します。この例では、毎月のサブスクリプション。

      Rokuのサブスクリプション間隔は他のストアとは異なることに注意してください。詳細については、サブスクリプション間隔のサポートサブスクリプションパッケージの使用ドキュメントのセクション
    5. [ 価格範囲 ] オプションでは、チャネル内製品の価格に基づいて価格帯を選択します。
  8. の中にトライアルとオファーセクションでは、チャネル内の製品戦略に応じて、選択できます無料トライアルに数値の期間を入力します試用期間フィールドを選択して選択します日々また月提出された時から。
  9. ページの下部に移動し、[ Cleared for sale ] チェックボックスをオンにし、[ 保存 ] ボタンをクリックします。
  10. 作成後、[ マイチャネル内製品の管理] ページに製品が表示されていることを確認します。
    提供するサブスクリプションのタイプに基づいて、追加のチャネル内製品をいくつでも作成できます。例：毎年または毎月。

チャネル内製品を Brightcove Beacon のチャンネルに関連付ける

チャネル内製品が作成されたら、Brightcove Beacon プラットフォームにアクセスして、新しいチャネル内製品をチャネルに関連付けます。

  1. トップメニューで [ コマース ] タブをクリックします。
  2. 左側のメニューから、[ パッケージ ] オプションを選択します。
  3. パッケージをクリックします。
  4. [ パッケージ ] タブで、[ 価格の詳細 ] セクションに移動します。
  5. [ 価格 ] フィールドに、チャネル内製品の価格を入力します。
  6. [ 通貨 ] フィールドで、通貨の種類を選択します。
  7. Roku ストア ID に、Roku 開発者プラットフォームで作成したチャネル内製品の識別子 ID を入力します。
  8. ページの下部に移動し、[ パッケージの編集 ] ボタンをクリックして、変更を保存します。
