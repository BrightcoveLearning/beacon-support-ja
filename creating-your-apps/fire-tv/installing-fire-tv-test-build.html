はじめに

このドキュメントでは、アプリをテストする前にアプリの提出プロセスを実行する必要がないように、テストビルドをインストールする方法について説明します。

Brightcove Beacon アプリをFire TVデバイスにインストールするには、次のものが必要です。
  • Fire TVデバイスで開発者モードが有効になります。
  • Androidデバッグブリッジ（adb）がコンピュータにインストールされています。

サブスクリプションやイベント支払いなど、アプリ内部の支払い方法をテストする場合は、 Live App Testing のインストール手順に従ってください。

プロセスの概要は次のとおりです。

  1. Amazon Fire TV端末でデバッグを有効にします。
  2. Androidデバッグブリッジ（adb）からFire TVデバイスに接続します。
  3. Brightcove Beacon アプリをインストールします。
  4. Brightcove Beacon アプリを実行します。

Amazon Fire TVでのデバッグの有効化

Fire TVデバイスでデバッグを有効にするには、以下の手順に従ってください。

  1. Fire TVのメイン画面で、[ 設定 ] を選択します。
  2. [ デバイス] (または [My Fire TV]) [> 開発者オプション] を選択します。
  3. [ ADB デバッグ ] をオンにします。
  4. [ 不明なソースからのアプリ ] をオンにします。

Androidデバッグブリッジ（adb）経由で接続する

アプリをインストール、テスト、デバッグするには、Android Debug Bridge（adb）を使用してコンピュータを Amazon Fire TV デバイスに接続する必要があります。adbを有効にする方法の詳細については、AmazonおよびAndroidのドキュメントに従ってください。

Brightcove Beacon アプリをインストールする

Amazon アプリストアに送信する前に Brightcove Beacon アプリをインストールしてテストするには、Android デバッグブリッジ（adb）を使用する必要があります。

独自のアプリをインストールする (Appstore の外) は、アプリをサイドローディングと呼ぶことがあります。

これらの手順を実行するには、すでに使用している必要があります adb 開発用コンピューターをFire TVデバイスに接続します。
  1. ターミナルまたはコマンドラインを開きます。
  2. 次のコマンドを入力します。ここで、path-to-apk-fileはアプリの APK へのファイルシステムパスです。 
    adb インストールパス-APK ファイル
  3. インストールが成功した場合、adb は次のようなメッセージで応答します。 
    764 KB / s（0.277sで217246バイト）pkg：/data/local/tmp/HelloWorld.apk成功
  4. デバイスにすでに存在するアプリを再インストールするには、-rオプションを使用してアプリを再インストールできます。 
    adb install -rpath-to-apk-file

Brightcoveの Beacon アプリの実行

サイドロードされたアプリは、[ 最近使った項目 ] 行と [ アプリ ] セクションの [ マイライブラリ ] 行の両方に表示されます。アプリは、[ 設定 ] メニューでも確認できます。

  1. Amazon Fire TVのメイン画面から、設定>アプリケーション>インストール済みアプリケーションの管理を選択します。
  2. Brightcoveの Beacon アプリを選択します。
  3. [ アプリケーションの起動 ] を選択します。

AmazonライブアプリテストサービスでのFire TVアプリのインストール

このセクションでは、Amazon Live アプリテストサービスで Fire TV アプリをインストールしてテストする方法について説明します。

サブスクリプションやイベント支払いなど、アプリの内部支払い方法をテストする場合は、このインストール手順を使用してください。

端末に Fire TV アプリをインストールするには、Brightcoveからのライブアプリテストの招待メールが必要です。

ライブアプリテストの招待メールが届かない場合は、こちらに従ってくださいリンクラベルの付いたオプションを選択して、Appstoreのマーケティングコミュニケーションを有効にします次のカテゴリからマーケティングメールを送ってくださいをクリックします保存する。

マーケティングコミュニケーションプリファレンスの設定後、変更は次のメール招待テストと後続のすべての招待テストから始まります。

ライブアプリテストに参加するための招待メールを受け取ったら、以下の手順を実行してください。

  1. 招待メールで、テストするマーケットプレイスのリンクをクリックします

    リンクをクリックすると、指定したマーケットプレイスの Brightcove Beacon アプリのリテールページが開きます。

  2. アプリ小売ページの [ 配信先: ] ドロップダウンメニューから、Brightcove Beacon アプリケーションをインストールするデバイスを選択します。
  3. テスターの現在のマーケットプレイスで特定のデバイスが使用できない場合、そのデバイスはドロップダウンメニューに表示されません。特定のマーケットプレイスでアプリがターゲットとするデバイスがない場合は、アプリの小売ページにあるドロップダウンメニューが [ 続行 ] ボタンに置き換えられます。
  4. 指定したデバイスに Brightcove Beacon アプリケーションをインストールします。アプリがターゲットとするAmazonアカウントに関連付けた端末の数だけテストできます。
  5. Amazon Fire TVのメイン画面から、設定>アプリケーション>インストール済みアプリケーションの管理を選択します。
  6. Brightcoveの Beacon アプリを選択します。
  7. [ アプリケーションの起動 ] を選択します。

Brightcove Beacon アプリのアンインストール

Fire TVデバイスからアプリをアンインストールするには、次の手順に従います。

  1. Fire TVのメイン画面から、設定>アプリケーション>インストール済みアプリケーションの管理を選択します。
  2. Brightcoveの Beacon アプリを選択します。
  3. アンインストール>アンインストールを選択します
