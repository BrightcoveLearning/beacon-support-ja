概要

以下のプラットフォームで顧客からの支払いを許可するには、Stripeアカウントが必要です。

  • Web
  • サムスンのスマートテレビ
  • ソニースマートTV
  • 東芝スマートテレビ

これは、WebアプリとすべてのスマートTVアプリにStripeが必要であることを意味します 以外 AndroidTVとFire TV。

次の手順では、Brightcove Beacon で Stripe を使用するプロセスの概要を説明します。

  1. 基本情報（メールアドレス、名前、パスワード）を入力してアカウントを作成します。
  2. ビジネスに関する情報を提供して、アカウントを有効にしてください。
  3. ストライプ API キーを取得します。
  4. Brightcoveの従業員に API キーを提供して、Brightcoveの Beacon エクスペリエンスに追加できるようにします。

Stripeアカウントの作成とアクティブ化

  1. Stripeのホームページを参照します。
  2. [ 今すぐ開始 ] ボタンをクリックします。
  3. 基本情報を入力してアカウントを作成します。
  4. Stripe ダッシュボードが表示されていることを確認します。
  5. Stripeアカウントの有効化オプションをクリックします。
  6. [ Start Now ] ボタンをクリックして、プロセスを開始します。
  7. メールアドレスを確認する必要があります。
  8. メールアドレスを確認した後、アカウントを有効にするために記入するフォームが表示されます。黄色でハイライトされているのは、フォームの主要なセクションです。
  9. フォームに入力したら、[ 申請を送信 ] ボタンをクリックします。

ストライプ API キーの取得

お客様のStripeアカウントとBrightcove Beacon 間の接続は、Stripe API キーを介して行われます。上記のダッシュボードのスクリーンショットから、ライブ API キーを取得するためのリンクが表示されます。アカウントがアクティベートされるまで、テストAPIキーしか取得できません。

ダッシュボードで、[ ライブ API キーを取得 ] をクリックし、指示に従ってダウンロードします。

ストライプをBrightcove Beacon に接続する

Stripe API キーを取得したら、Brightcoveの Beacon エクスペリエンスを設定しているBrightcoveの担当者にそのキーを渡します。Stripe API キーを Brightcove Beacon エクスペリエンスに関連付けて、Stripe をアプリで使用するように設定すると、お客様のアカウントに関連付けられます。
