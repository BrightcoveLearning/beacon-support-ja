新しいクーポンを作成する手順

Brightcove Beacon で新しいクーポン/バウチャーを作成するには、次の手順に従います。

  1. Brightcoveの Beacon UI で、コマースタブをクリックします。
  2. 左側のメニューで、[ Stripe によるクーポン ] を選択します。
  3. 新しいバウチャーを追加ボタンをクリックします。
  4. [ ID ] フィールドに、ユーザーが使用するコードを入力します。
  5. [ 名前 ] フィールドには、請求書に表示できるわかりやすい名前を使用します。
  6. [ 期間 ] フィールドに、クーポンの期間を月単位で入力します。
  7. 割引タイプでは、通貨またはパーセントで割引を指定するオプションがあります。たとえば、割引が 3 か月無料の場合、期間を 3 か月で繰り返し、割引の種類を 100% に設定します。
  8. [ 制限 ] セクションでは、クーポンを使用する償還回数または日付を制限できます。たとえば、8 月末までしかクーポンを使用できない場合や、最初の 100 人だけがクーポンを取得するとします。
  9. [ 関連するパッケージ/サブスクリプション ] セクションで、クーポンで使用できるサブスクリプションを指定します。サブスクリプションを選択しない場合、そのサブスクリプションはすべてのパッケージサブスクリプションに適用されます。
次の点に注意してください
  • クーポン/バウチャーが作成されると、登録ユーザーはサブスクリプションを購入する際にクーポン/クーポンを適用できます。
