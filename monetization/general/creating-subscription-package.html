サブスクリプションパッケージの作成

このトピックでは、を使用してビデオサブスクリプションパッケージを構成する方法を学習します。コマースBrightcove Beacon のタブ。サブスクリプションパッケージは、SVOD (サブスクリプションビデオオンデマンド) の収益化を実装します。サブスクリプションパッケージをビデオに適用する方法については、サブスクリプションパッケージの使用資料。

概要

サブスクリプションパッケージは、特定のビデオセットをアプリで購入する方法を定義します' 顧客。パッケージを作成するときに、たとえば、次のように構成できます。

  • 請求頻度
  • 広告がサポートされているかどうか、および使用する広告構成を設定します
  • お支払い方法
  • パッケージを使用してユーザーが同時に再生できるストリームの最大数
  • パッケージを購入できる開始日と終了日
  • パッケージ内のビデオを表示するために許可されたデバイスと地理的位置

パッケージを作成する手順

次の手順を実行して、新しいパッケージを作成します。

  1. Brightcove Beaconで、コマースタブ、必ずパッケージ左側のナビゲーションでが選択されている場合は、新しいパッケージを追加する。
  2. にパッケージタブ（下のスクリーンショット）で、以下を完了します。
    • 名前 :ビデオのVideoCloudカスタムフィールドで使用されるパッケージの名前。
    • 請求頻度：このパッケージの請求頻度を選択してください。
    • 試用期間：このパッケージの試用期間。入る0試用期間なし。
    • 状態：パッケージを公開して使用できるようにするかどうかを選択します。少なくとも1つの支払い方法が選択されるまで、パッケージを公開できないことに注意してください。
    • 広告：このパッケージで広告を使用するかどうかを選択します。
    • 広告プロバイダー：前の手順で広告を使用することを選択した場合は、広告プロバイダーを選択してください。このドロップダウンには、にリストされているアドバタイズメント構成が表示されます。広告Brightcove Beacon のセクション。
    • 使用するお支払い方法：このパッケージで使用する支払い方法を選択します。選択した各方法について、支払い方法のリストの下にセクションが開きます。これは、選択した方法ごとに完了する必要があります。以下のスクリーンショットはストライプ（黄色）そしてアップルストア（赤）が選択されています。
  3. [ Create Package ] ボタンをクリックして、現在のパッケージ設定を保存します。
  4. にテキストデータタブで、以下を完了します。
    • パック名：ユーザーに表示される名前。
    • 見出し：たとえば、ユーザーに表示される見出し ホリデースペシャル！ 。
  5. [ Create Package ] ボタンをクリックして、現在のパッケージ設定を保存します。
  6. に非テキストデータタブで、パッケージ名の横に使用する画像を入力します。
  7. [ Create Package ] ボタンをクリックして、現在のパッケージ設定を保存します。
  8. ザ・コンテンツ現在、タブは使用できません。
  9. にストリームタブで、以下を完了します。
    • 最大ストリーム：同時にストリーミングできる接続デバイスの最大数。
    • 最大デバイス：アクティブな接続を持つことができる接続済みデバイスの最大数。
      上記の2つの設定もグローバルに設定されているため、このパッケージのグローバル設定を上書きする場合にのみ、これらの設定を使用してください。
    • 高度なストリーム：これをに設定しますはいこのパッケージが設定する権利をカスタマイズしたい場合。利用可能な変更を次のスクリーンショットに示します。
  10. クリック新しい権利を追加する新しく構成され、カスタマイズされた権限を保存するためのボタン。
  11. [ Create Package ] ボタンをクリックして、現在のパッケージ設定を保存します。

ストライプクーポンの作成

クーポンを作成して、パッケージの購入時に適用できる固定金銭割引またはパーセンテージ割引のいずれかの割引を提供できます。

  1. Brightcove Beaconで、コマースタブをクリックし、をクリックしますStripeによるクーポン最後にクリックします新しいバウチャーを追加する。
  2. に新しいクーポンを作成するフォーム（下のスクリーンショット）で、以下に記入します。
    • Id：視聴者がクーポンを使用するためにBrightcove Beacon アプリに入力するコード
    • 名前 :請求書に表示できるクーポンのわかりやすい名前
    • デュレーション：から選ぶ一度、永遠にまた繰り返し ;
      もしも繰り返しを選択し、繰り返し使用できる月数を設定します。
    • にとって割引タイプ次のいずれかを選択します。
      • 金額：クーポンの値と通貨を入力します
      • パーセンテージ：クーポンの割引率を入力してください

      例：割引が3か月無料の場合は、デュレーションに繰り返し 3ヶ月で設定します割引タイプとパーセンテージ 100％の

    • 制限日（UTC）：クーポンを使用できる最後の日付を選択します
    • 最大償還：クーポンの最大償還数を入力します。たとえば、50人の顧客だけがクーポンを引き換えることができます
    • パッケージを選択：クーポンを適用できるパッケージを選択します。パッケージを選択しない場合、クーポンはすべてのパッケージに適用できます
      例：クーポンは、年間サブスクリプションを購入した場合にのみ有効です

  3. 必ずクリックしてください保存する新しいクーポンを保存するボタン。
