広告設定は、広告呼び出し、タイミング、 Beacon 、その他の広告設定オプションなど、広告の再生に関するさまざまな側面を定義します。サーバー側広告設定では、動画オンデマンド（VOD）動画やライブイベント中にリクエストできる広告を定義します。サーバーサイド広告は、ブライトコーブのサーバーサイド広告挿入（SSAI）テクノロジーを活用し、次の主な機能を提供します。

  • 広告はサーバー側のオンデマンド動画ストリームまたはライブ動画ストリームに縫い付けられるため、広告ブロッカーの影響を受けません
  • ライブイベントの場合、エンコーダから送信されたキューポイントに広告が挿入されます。または Live モジュール Control Room を使用してインスタントキューポイントを作成できます。
  • 未使用の広告時間を埋めるように塗りつぶしスレートを構成できます

サーバーサイド広告の詳細については、「 Brightcove Player でのサーバーサイド広告の実装」を参照してください。

ノート：
  • SSAI を使用するには、Video Cloud アカウントをダイナミック配信に設定し、SSAI を有効にする必要があります。詳細はアカウントマネージャーにお問い合わせください。
  • ブライトコーブ Live SSAI の VMAP は現在サポートされていません。
  • SSAI 広告にはプリフェッチ機能はありませんが、広告ポジションロジックは広告プロバイダーに委任されます。動的配信では、すべての追加コールが前もってきます。広告プロバイダから広告を取得し、それが私たちのシステムでトランスコードされている場合は、ストリームに挿入します。

[サーバー側広告設定] ページは、ライブ広告と VOD 広告の設定を管理するために使用します。[サーバー側広告設定] ページにアクセスするには、管理者ロールを持つユーザーである必要があります。

[サーバーサイド広告設定] ページにアクセスするには:

  1. Studioにログインします。
  2. ナビゲーションヘッダーで、[ 管理者 ] をクリックし、[ サーバーサイド広告設定 ] をクリックします。すべての広告設定の一覧が表示されます。

新しい広告設定を作成するには、[ 広告設定の作成 ] をクリックし、作成する広告の種類をクリックします。

  • ビデオオンデマンド (VOD)
  • ライブ

動画オンデマンド広告の設定を作成する

ノート：サーバーサイド広告の設定は、サーバーサイド広告挿入 API を使用して管理することもできます。詳細については、を参照してください。 Video Cloud SSAI Ad Config APIトピック。

動画オンデマンド（VOD）広告の設定を追加するには、次の手順に従います。

  1. [サーバー側広告設定] ページで、[ 広告設定の作成] [> VOD ] をクリックします。
  2. 「名前」を入力します。
  3. 広告サーバーの応答を選択します。
    • DFP 広告ルール
    • DFP VMAP
    • スマートXML
    • 広大3.0
  4. 広告タグを入力します。
  5. サーバー側 Beacon を無効にするには、[サーバー側 Beacon を無効にする] をオンにします。無効にすると、SSAI は Beacon をサーバー側で起動せず、すべての Beacon を VMAP 出力に含めます。
  6. [保存]をクリックします。

広告設定を編集するには、広告名をクリックして変更し、[ 保存 ] をクリックします。広告設定を削除するには、広告の横にあるチェックボックスをオンにし、[ 削除 ] をクリックします。広告設定を複製するには、広告の横にあるチェックボックスをオンにし、[ 複製 ] をクリックします。

VOD 広告設定を作成したら、Players モジュールを開き、プレーヤーをクリックしてプロパティを表示します。サーバー側（SSAI）広告を有効にするプロパティが有効な場合は、広告構成を選択できます。プレーヤー広告の設定については、「 Players モジュールを使用したプレーヤー広告の設定 (外部リンク) 」を参照してください。

ライブ広告の設定を作成する

ノート：サーバー側のライブ広告の設定は、ライブ API を使用して管理することもできます。詳細については、「ブライトコーブ Live API」を参照してください。サーバーサイド広告挿入（SSAI）。

ライブ広告の設定を追加するには、次の手順に従います。

  1. [サーバーサイド広告設定] ページで、[ > ライブ広告設定の作成 ] をクリックします。
  2. 「名前」を入力します。
  3. 広告サーバーの応答を選択します。
    • 広大
    • DFP (広告ルール)
    • スマートXML
  4. 広告タグを入力します。
  5. [保存]をクリックします。

広告設定を編集するには、広告名をクリックして変更し、[ 保存 ] をクリックします。広告設定を削除するには、広告の横にあるチェックボックスをオンにし、[ 削除 ] をクリックします。広告設定を複製するには、広告の横にあるチェックボックスをオンにし、[ 複製 ] をクリックします。

ライブ広告構成を作成した後、サーバーサイド広告挿入を有効にする (SSAI) プロパティを有効にしてイベントを作成すると、コントロールルームで広告構成を選択できます。ライブイベントの作成については、「ライブモジュールを使用したライブイベントの作成と管理」を参照してください。
